TPC4
Manipulação de Listas

Neste TPC, é oferecido um menu com várias interações possíveis:

(1) Criar Lista 
(2) Ler Lista
(3) Soma
(4) Média
(5) Maior
(6) Menor
(7) estaOrdenada por ordem crescente
(8) estaOrdenada por ordem decrescente
(9) Procura um elemento
(0) Sair

Assim, cada opção engloba a sua função, com seu propósito:

1- Uma lista é criada, com número aleatório de 1 a 100 elementos, que podem variar entre 0 e 100

2- Uma lista é criada, com base nos inputs do utilizador

3- É devolvida a soma de todos os elementos de uma lista

4- É devolvida a média de todos os elementos de uma lista

5- É devolvida o maior de todos os elementos de uma lista

6- É devolvida o menor de todos os elementos de uma lista

7- Verifica se uma lista está ordenada em sentido crescente

8- Verifica se uma lista está ordenada em sentido decrescente

9- Verifica se um elemento especificado pelo utilizador se encontra numa lista, devolvendo a sua posição se este estiver presente, ou "-1" se não estiver
