TPC7
Gestão e Apresentação de Tabelas Meteorológicas

Este TPC consiste na gestão de dados respetivos a listas de datas e dados meteorológicos, como temperaturas e precipitação.

Para este TPC foram utilizados dicionários. Para execução de muitas das funções, foram utilizadas as funcionalidades de "keys" dos dicionários para fins de organização e acesso de dados.

Neste TPC, é apresentado um menu com várias funcionalidades:

1) Ver temperaturas médias
2) Guardar uma tabela meteorológica
3) Carregar uma tabela meteorológica
4) Ver temperatura mínima
5) Ver amplitudes térmicas
6) Ver precipitação máxima
7) Ver dias com precipitação acima de um valor
8) Ver dias com precipitação abaixo de um valor
9) Construir gráficos de temperatura e precipitação
0) Sair

1- Devolve a temperatura média para cada data especificada na tabela (lista) inserida

2- Guarda uma tabela num formato .txt, de forma a garantir a sua fácil leitura e extração

3- Importa uma tabela, a partir de um ficheiro, guardando-a na memória local para futura utilização

4- Devolve a temperatura mínima para cada data especificada na tabela (lista) inserida

5- Devolve a amplitude térmica para cada data especificada na tabela (lista) inserida

6- Devolve a precipitação máxima para um conjunto de datas especificadas na tabela (lista) inserida

7- Devolve o número de dias com precipitação acima de um valor arbitrário para todas as datas especificadas na tabela (lista) inserida

8- Devolve o maior número de dias consecutivos com precipitação abaixo de um valor arbitrário para todas as datas especificadas na tabela (lista) inserida

9- Apresenta um menu com duas funções:

(1) Gráfico de temperaturas
(2) Gráfico de pluviosidade 

1- Apresenta um gráfico de linhas das temperaturas máximas e mínimas de todas as datas especificadas na tabela (lista) inserida

2- 1- Apresenta um gráfico de barras das precipitações de todas as datas especificadas na tabela (lista) inserida
